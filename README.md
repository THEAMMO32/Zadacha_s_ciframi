# Zadacha_s_ciframi

## Описание задачи

Решение задачи поиска k-й цифры в бесконечной последовательности, составленной путём приписывания друг к другу всех натуральных чисел:

```
1234567891011121314151617...
```

Программа находит цифру, которая стоит на k-й позиции этой последовательности (нумерация с единицы).

## Алгоритм решения

Решение основано на математическом подходе без построения всей строки:

1. **Определение группы чисел**: Числа разбиваются на группы по количеству цифр (1-значные, 2-значные, и т.д.)
   - 1-значные числа (1-9): 9 чисел × 1 цифра = 9 позиций
   - 2-значные числа (10-99): 90 чисел × 2 цифры = 180 позиций
   - 3-значные числа (100-999): 900 чисел × 3 цифры = 2700 позиций
   - И так далее...

2. **Поиск конкретного числа**: После определения группы вычисляется конкретное число и позиция цифры внутри него

3. **Извлечение цифры**: Из найденного числа извлекается нужная цифра

## Примеры

### Пример 1
```
k = 5
Последовательность: 12345...
Цифра на позиции 5 → 5
```

### Пример 2
```
k = 12
Последовательность: 123456789101112...
Цифра на позиции 12 → 1 (первая цифра числа 11)
```
